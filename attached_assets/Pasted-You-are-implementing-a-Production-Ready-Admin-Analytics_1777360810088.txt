# 📈 GRAPH SECTION

## 1. Revenue Over Time (Line Chart)

Group by date:

```js id="rev_time"
{
  $group: {
    _id: {
      $dateToString: { format: "%Y-%m-%d", date: "$createdAt" }
    },
    revenue: { $sum: "$totalAmount" }
  }
}
```

---

## 2. Orders Over Time (Line/Bar)

```js id="orders_time"
{
  $group: {
    _id: {
      $dateToString: { format: "%Y-%m-%d", date: "$createdAt" }
    },
    count: { $sum: 1 }
  }
}
```

---

# 🔥 TOP PRODUCTS

```js id="top_products"
[
  { $unwind: "$products" },
  {
    $group: {
      _id: "$products.productId",
      totalSold: { $sum: "$products.quantity" }
    }
  },
  { $sort: { totalSold: -1 } },
  { $limit: 5 }
]
```

Join with products collection for:

* name
* image

---

# ⚠️ STOCK ALERTS

From `products`:

```js id="stock"
{ stock: { $lte: 5 } }
```

Show:

* low stock
* out of stock

---

# 🧾 RECENT ORDERS

```js id="recent"
.find()

.limit(5)
```

Display:
