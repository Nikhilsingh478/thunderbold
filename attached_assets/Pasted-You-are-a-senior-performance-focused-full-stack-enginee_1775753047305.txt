# 🔥 PRIORITY 1: FIX RENDERING FLOW (CRITICAL)

## Problem:

UI is waiting for API data before rendering.

## Task:

Render UI IMMEDIATELY, then hydrate with data.

### Steps:

1. In homepage (`Index.tsx` or equivalent):

```ts

}, []);
```

3. UI should render BEFORE data arrives

# 🔥 PRIORITY 3: PARALLELIZE API CALLS

## Problem:

Sequential API calls slow everything

## Fix:

Replace:

```js
await fetchProducts();
await fetchCategories();
```

With:

```js
await Promise.all([
  fetchProducts(),
  fetchCategories()
]);
```
