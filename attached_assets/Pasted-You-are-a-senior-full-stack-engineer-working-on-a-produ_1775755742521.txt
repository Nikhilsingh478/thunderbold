# 🚨 PRIORITY 1: AUTO-OPTIMIZE CLOUDINARY URLs

## Problem:

Currently raw Cloudinary URLs may be used.

## Task:

Create a utility function to transform ALL Cloudinary URLs before rendering.

---

## Step 1: Create utility file

`src/lib/cloudinary.ts`

---

## Step 2: Add function

```ts
export function optimizeCloudinaryUrl(
  url: string,
  width: number = 800
): string {
  if (!url || !url.includes("res.cloudinary.com")) return url;

  return url.replace(
    "/upload/",
    `/upload/f_auto,q_auto,w_${width}/`
  );
}
```

---

## Step 3: Usage

Replace ALL image usages:

```tsx
<img src={product.image} />
```

WITH:

```tsx
<img src={optimizeCloudinaryUrl(product.image, 800)} />
```
